\# Project Management System



\## 📌 Project Overview

The Project Management System is a full-stack web application designed to help teams manage projects, tasks, and workspaces efficiently.

It allows users to create projects, assign tasks, collaborate with team members, and track progress in real time.



This system demonstrates core Database Management System concepts such as normalization, transactions, concurrency control, triggers, views, and constraints.



\---



\## 🚀 Features



\- User Authentication using Clerk

\- Workspace creation and management

\- Project creation and tracking

\- Task assignment and management

\- Team collaboration

\- Activity logging

\- Email notifications

\- Project analytics dashboard

\## ⚙️ Setup Instructions



\### 1️⃣ Clone the repository





git clone https://github.com/Nowfid-shaik/project-management-system.git





\### 2️⃣ Install dependencies





cd server

npm install



cd ../client

npm install





\### 3️⃣ Run Backend





cd server

npm run server





\### 4️⃣ Run Frontend





cd client

npm run dev
